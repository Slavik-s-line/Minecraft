# Minecraft

![logo](https://upload.wikimedia.org/wikipedia/uk/thumb/3/32/Minecraft_logo.svg/2560px-Minecraft_logo.svg.png)


## Get started
- Clone this repositories
```
git clone https://github.com/Slavik-s-line/Minecraft.git
```
- Install Panda3d
```
pip install Panda3D
```
- Run project
```
python3 game.py
```
